# Codify

A simple way to play around with (semi-random) code databases while easily managing their data.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requirements.

```bash
pip install -r reqs.txt
```

## Usage

```bash
python codegen.py
```
