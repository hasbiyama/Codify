# Codify

A simple way to play with (semi-random) code generator databases while easily managing their data.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requirements.

```bash
pip install -r reqs.txt
```

## Usage

```bash
python codegen.py
```
